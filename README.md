## Sierpinski Carpet project

A Polish mathematician, Waclaw Sierpinski defined a special fractal in 1915. To draw these fractals, it starts with a full black equilateral triangle. In this case, I am working with squares instead of triangles. Obtain three new full black squares and perform the same procedure. The algorithm draws the fractals. When the algorithm reaches a maximum number of recursive calls, the main process draws all lines in the data structure to obtain a visual representation of the fractal. 

It's implemented as a graphical interface, and it can be scaled, zoomed, and moved. Initially, it scales to fit all points in the viewport.

The program can be run in different modes (sequential, parallel, distributed) by specifying a parameter. The program measures the run-time needed to complete.
